## 约定与术语表

| 术语 | 说明 |
|------|------|
| **StyleFlow** | 项目名称，服装设计-生产协同平台 |
| **Django Ninja** | Django 的 RESTful API 框架，支持类型注解、自动 OpenAPI 文档 |
| **ERP** | 企业资源计划系统，本文指代服装企业内部管理系统 |
| **BOM** | 物料清单（Bill of Materials），服装款式所需物料明细 |
| **工艺单 / Tech Pack** | 包含款式图、尺寸表、工序说明的生产指导文档 |
| **核工价** | 对服装加工各工序的标准工时和工费进行核定 |
| **VTON** | 虚拟试衣（Virtual Try-On），将服装图合成到人物图 |
| **SD** | Stable Diffusion，开源文生图模型 |
| **Celery Canvas** | Celery 的 workflow 原语（chain, group, chord） |
| **SSE** | Server-Sent Events，服务端推送，用于任务进度通知 |
