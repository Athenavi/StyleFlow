# StyleFlow Brand Guidelines

## Logo

`logo.svg` — Purple gradient circle with "SF" letters and sparkle accents.

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| **Primary** | `#667eea` | Buttons, links, primary actions |
| **Secondary** | `#764ba2` | Gradient accent, hover states |
| **Success** | `#52c41a` | Completed status, approval |
| **Warning** | `#faad14` | Pending, attention needed |
| **Error** | `#ff4d4f` | Rejected, errors, deletion |
| **Bg Light** | `#ffffff` | Light mode background |
| **Bg Dark** | `#141414` | Dark mode background |

## Typography

- **Brand Font**: Georgia / serif (logo)
- **UI Font**: system-ui, -apple-system, sans-serif

## Logo Usage

- Always use the SVG for best quality
- Minimum clear space: 16px around the logo
- On light backgrounds: use full color logo
- On dark backgrounds: use white version

## Tone of Voice

- **English**: Professional, clear, minimal
- **Chinese**: 专业、清晰、简洁

*"Let inspiration flow naturally, let processes run smoothly."*
*"让灵感自然涌动，让流程自如流转。"*
